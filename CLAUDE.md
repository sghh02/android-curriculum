# Claude 向けカリキュラムリポジトリ運用ルール

このリポジトリは **学習用カリキュラムコンテンツ** を管理するためのものです。  
本リポジトリ内のファイルは、Webアプリケーションによって **そのまま表示** されます。

Claude は、本リポジトリ内のファイルを **作成・編集する役割** として使用される場合があります。  
その際は、以下のルールを **必ず厳守** してください。

---

## リポジトリ構成

- `index.json`  
  - カリキュラム全体の構造定義ファイル
  - 章・レッスンのメタ情報のみを記載
  - 厳密に正しい JSON であること

- `*.md`  
  - 各レッスンの本文
  - Webアプリで **加工なしに直接描画** される

---

## index.json のルール（最重要）

- **必ず有効な JSON のみを出力すること**
- コメントを入れない
- ```json などのコードブロックで囲まない
- 説明文や補足文を一切書かない
- 末尾カンマを使わない
- `JSON.parse()` でそのまま読み込めること

### 必須構造

```json
{
  "title": "コース名",
  "chapters": [
    {
      "title": "チャプター名",
      "items": [
        {
          "id": "一意なID",
          "title": "レッスン名",
          "path": "relative/path/to/file.md"
        }
      ]
    }
  ]
}
