# Android Studioのインストール

Androidアプリ開発には、公式IDEである **Android Studio** が必要です。

## インストール手順
1. [公式サイト](https://developer.android.com/studio)からインストーラーをダウンロードします。
2. インストーラーを実行し、標準設定（Standard）で進めます。

### 注意点
> エミュレータを動かすには、PCのBIOS設定で仮想化技術（VT-x / AMD-V）が有効になっている必要があります。

// 初めてのコード
fun main() {
    println("Hello, Android!")
