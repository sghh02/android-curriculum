# 総合演習：TODOアプリを作る

これまで学んだ内容を活かして、シンプルなTODOアプリを作成します。

## 目標
- Compose + MVVM + Roomを組み合わせる
- 実際のアプリ構成に近い形で実装する

## 要件
- TODOの追加 / 完了 / 削除
- ローカルに保存（Room）
- Empty/Loading/Errorの状態管理

## 推奨構成
- `ui/` : Composable
- `viewmodel/` : ViewModel
- `data/` : Repository / Room / API

## 追加チャレンジ
- ダークモード対応
- 通知機能で期限をリマインド
- Jetpack Composeのアニメーション追加
